# Cloud 9 by atoffettidev

Cloud 9 is a Nextjs application sitting behind a Nginx web server backed by a Nodejs server api. Cloud 9 is a work space and communication hub for atoffettidev. This is where i'll be show casing web development projects and blogs.

## Goals

Considerations for Cloud 9 include optimization, adherance to best practices, use of modern development tools, minimal external imports, light weight packaging, accessibility, modern design and a hope to contribute to the development community with Cloud 9's blog.

### Optimization

Nextjs is the choice front end for Cloud 9 for it's renowned speed and performance. Adding to that performance in a production environment Cloud 9 sits behind a Nginx server enhancing the applications ability to perform in real time and handle concurrent requests.
The application only imports trusted packages when neccessary. In most other cases for example Cloud 9's animations, handling is achieved with custom local modules.

### Modern Design

Modern design theory and UX/UI considerations are at the fore front of Cloud 9's development process and will evolve over time.

## Growth

Over time features and upgrades will be integrated in Cloud 9's infastructure, it is built to scale and evolve. If you would like to keep up with the life cycle of Cloud 9 please reach out, that's what Cloud 9 is here for!
